Estrutura dos diretórios e Fontes

 

C:\MGFinanc \BASE    -> Base de dados               

    \IMAGES    -> Imagens Utilizadas na Compilação

 

F_MENU.EXE    -> Aplicativo

FINANC.INI    -> Arquivo INI

 

F_menu.Prg    -> Programa Principal        -> Documentado

F_Login.Prg    -> Login no Sistema        -> Documentado

F_Acesso.Prg    -> Cadastro de Usuários        -> Documentado

F_Clientes.Prg    -> Cadastro de Clientes        -> Documentado

F_Fornecedores    -> Cadastro dos Fornecedores    -> Idêntico à F_Clientes.Prg

F_CadGen.Prg    -> Função de Cadastros Genérico    -> Documentado

F_Funcoes.Prg    -> Funções do Sistema        -> Documentado

F_Datas.Prg    -> Funções de Datas        -> Documentado

F_Pagar.Prg    -> Contas à Pagar            -> Documentado

F_Receber    -> Contas a Receber        -> Idêntico à F_Pagar.Prg

       

F_Menu.RC    -> Arquivo de Resource do Sistema

F_Sistema.CH    -> Arquivo CH do Sistema
